# Inventory & Order Management - Full Project

This zip contains two folders:
- backend (Spring Boot, MySQL)
- frontend (React + Vite)

Follow README.md in each folder. Update backend `application.properties` with your MySQL password.

Quick steps:
1. Create MySQL DB: CREATE DATABASE inventory_db;
2. Start backend: mvn spring-boot:run (from backend folder)
3. Start frontend: npm install && npm run dev (from frontend folder)
